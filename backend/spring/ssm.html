# SSM整合

本质：Spring接管一切（将框架核心组件交给Spring进行IoC管理），代码更加简洁。

  • SpringMVC管理web相关组件
  • Spring管理业务层、持久层、以及数据库相关（DataSource,MyBatis）的组件
  • SSM整合最终就是编写IoC配置文件

# 理解

在SSM整合过程中会创建两个IOC容器，其实一个容器也是可以，但是所有的配置都放在一个xml文件中，会导致配置文件太长难以维护

容器名创建类盛放组件
web容器(子)DispatcherServletweb相关组件（controller,springmvc核心组件）
root容器(父)ContextLoaderListener业务和持久层相关组件（service,aop,tx,dataSource,mybatis,mapper等）

这里我们会创建三个配置文件，分别对应controller、service、mapper三层

配置文件对应层次配置信息
spring-mvc.xmlcontroller层controller
handlerMapping, handlerAdapter
静态资源处理器, 视图解析器, 文件上传处理, 拦截器等
spring-service.xmlservice层service
aop (advice, pointcut, 开启aop注解支持 )
tx (事务管理器, 开启tx注解支持 )
spring-mapper.xmlmapper层druidDataSource
mybatis的sqlSessionFactory (数据库信息, mapper信息, mybatis的配置 [日志,驼峰式映射,resultMap嵌套映射])
mapper代理对象

总结

自己的理解(不一定全部正确)

  • servlet启动时会有三个进行初始化，分别为servlet的init, Filter的init, Listener的init，这三个当中Listener的优先级最高，也正是我们的root(父容器)被初始化的地方

  • root容器初始化时，会加载我们配置的spring-service.xml和spring-mapper.xml的配置文件，因为我们的service层和mappper层相关的组件会放到root容器中

  • 初始化完成后，root容器会放在全局最大的共享域ServletContext中

  • 接下来就是web容器(子容器)的初始化，是在servlet的init方法中被初始化的，这时会在全局最大的共享域中拿到root容器，使用setParent方法，认root容器为父容器，是因为以后我们要使用controller层调用service层，认为父容器之后可以调用父容器中的方法

  • 这样两个IOC容器就初始化完成了，下面我们再来了解一下sqlSessionFactory的生成

  • sqlSessionFactory会通过SqlSessionFactoryBean中的getObject()方法给我们返回一个SqlSessionFactory对象，这期间还会把我们相关的配置一起加载

  • 下面再说一下mapper的代理对象的生成

  • 因为FactoryBean一次只能生成一个对象但是我们的mapper接口会有多个，这时利用MapperScannerConfigurer中的方法遍历我们的接口文件使用FactoryBean依次生成

# spring-service

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xmlns:context="http://www.springframework.org/schema/context"
       xmlns:aop="http://www.springframework.org/schema/aop" xmlns:tx="http://www.springframework.org/schema/tx"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd http://www.springframework.org/schema/context https://www.springframework.org/schema/context/spring-context.xsd http://www.springframework.org/schema/aop https://www.springframework.org/schema/aop/spring-aop.xsd http://www.springframework.org/schema/tx http://www.springframework.org/schema/tx/spring-tx.xsd">
    
    <!--
        service
        aop
            advice
            pointcut
            开启aop注解支持
        tx
            事务管理器
            开启tx事务支持
    -->

    <!--扫描包让注解生效-->
    <context:component-scan base-package="com.atcode.service, com.atcode.advice, com.atcode.pointcut" />

    <!--开启aop注解支持-->
    <aop:aspectj-autoproxy />

    <!--配置事务管理器-->
    <bean id="transactionManager" class="org.springframework.jdbc.datasource.DataSourceTransactionManager">
        <!--赋值 druidDataSource-->
        <property name="dataSource" ref="dataSource" />
     </bean>
    <!--开启事务注解支持-->
    <tx:annotation-driven transaction-manager="transactionManager" />
</beans>

# spring-mapper

jdbc.properties配置文件

jdbc.driver = com.mysql.cj.jdbc.Driver
jdbc.url = jdbc:mysql://localhost:3306/mybatis-example
jdbc.username = root
jdbc.password = root

spring-mapper.xml配置文件

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xmlns:context="http://www.springframework.org/schema/context"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd http://www.springframework.org/schema/context https://www.springframework.org/schema/context/spring-context.xsd">

    <!--
        druidDatasource

        mybatis
            sqlSessionFactory
                数据库信息
                Mapper信息
                设置信息
                    日志
                    驼峰映射
                    resultMap嵌套映射
                别名信息
                插件信息
            Mapper代理对象
    -->

    <!--加载数据库的配置文件-->
    <context:property-placeholder location="classpath:jdbc.properties"/>
    <!--配置druidDataSource-->
    <bean id="dataSource" class="com.alibaba.druid.pool.DruidDataSource">
        <property name="driverClassName" value="${jdbc.driver}"/>
        <property name="url" value="${jdbc.url}"/>
        <property name="username" value="${jdbc.username}"/>
        <property name="password" value="${jdbc.password}"/>
    </bean>

    <!--配置mybatis中的sqlSessionFactory-->
    <bean id="sqlSessionFactory" class="org.mybatis.spring.SqlSessionFactoryBean">
        <!--配置数据库 使用druidDataSource-->
        <property name="dataSource" ref="dataSource"/>
        <!--指明我们的mapper文件所在位置-->
        <property name="mapperLocations" value="classpath:mappers/*.xml"/>
        <!--mybatis的设置信息-->
        <property name="configuration">
            <bean class="org.apache.ibatis.session.Configuration">
                <!--开启驼峰式映射-->
                <property name="mapUnderscoreToCamelCase" value="true"/>
                <!--开启嵌套resultMap自动映射-->
                <property name="autoMappingBehavior" value="FULL"/>
                <!--配置logback日志输出-->
                <property name="logImpl" value="org.apache.ibatis.logging.slf4j.Slf4jImpl"/>
            </bean>
        </property>
        <!--起别名 所在的包-->
        <property name="typeAliasesPackage" value="com.atcode.pojo"/>
        <!--插件的配置-->
        <property name="plugins">
            <!--分页插件-->
            <bean class="com.github.pagehelper.PageInterceptor">
                <property name="properties">
                    <!--配置MySQL数据库-->
                    <value>helperDialect=mysql</value>
                </property>
            </bean>
        </property>
    </bean>

    <!--根据接口生成mapper的代理对象-->
    <bean class="org.mybatis.spring.mapper.MapperScannerConfigurer">
        <!--mapper接口所在的包-->
        <property name="basePackage" value="com.atcode.mapper"/>
    </bean>
</beans>

# IOC容器初始化

web.xml配置文件

<?xml version="1.0" encoding="UTF-8"?>
<web-app xmlns="http://xmlns.jcp.org/xml/ns/javaee"
	xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
	xsi:schemaLocation="http://xmlns.jcp.org/xml/ns/javaee http://xmlns.jcp.org/xml/ns/javaee/web-app_4_0.xsd"
	version="4.0">

	<!--
		ioc容器初始化
		ContextLoaderListener
		DispatcherServlet
	-->

	<!--设置全局变量 加载配置文件-->
	<context-param>
		<param-name>contextConfigLocation</param-name>
		<param-value>classpath:spring/spring-service.xml, classpath:spring/spring-mapper.xml</param-value>
	</context-param>

	<!--root容器初始化(父容器) 是在ContextLoaderListener中初始化-->
	<listener>
		<listener-class>org.springframework.web.context.ContextLoaderListener</listener-class>
	</listener>

	<servlet>
		<!--web容器初始化(子容器) 是在DispatcherServlet中初始化-->
		<servlet-name>ds</servlet-name>
		<servlet-class>org.springframework.web.servlet.DispatcherServlet</servlet-class>
		<!--加载配置文件-->
		<init-param>
			<param-name>contextConfigLocation</param-name>
			<param-value>classpath:spring/spring-mvc.xml</param-value>
		</init-param>
		<!--配置文件提前加载-->
		<load-on-startup>1</load-on-startup>
	</servlet>
	<!--拦截一切路径-->
	<servlet-mapping>
		<servlet-name>ds</servlet-name>
		<url-pattern>/</url-pattern>
	</servlet-mapping>
</web-app>
