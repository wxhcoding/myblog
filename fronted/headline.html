# 登录

用户在客户端输入用户名密码并向后端提交,后端根据用户名和密码判断登录是否成功,用户有误或者密码有误响应不同的提示信息

反射

  • 首先这里需要用到反射的相关知识，因为后端使用servlet去处理请求，涉及到请求地址的匹配问题，一个controller文件中会包含好几个请求地址，这里就有了请求地址的匹配问题，而我们现在使用反射的知识点来解决这个问题。
  • 首先获取到请求地址中的最后一个单词，使用此单词去匹配对应的执行方法，这就需要我们定义的方法要和请求地址中的最后一个单词相匹配，获取到之后，通过反射读取到同名的方法，也就是下面第三步的代码，通过setAccessible这个属性，可以忽略方法的访问权限，再通过invoke去调用对应的方法。
  • 其中method.invoke(this, req, resp);这三个传递的参数分别为，this表示是的是使用当前的这个类去调用这个类下面的某个方法，后面两个参数为传递过去的两个参数。
@WebServlet(value = "/user/*")
public class NewsUserServlet extends HttpServlet {
    NewsUserService service = new NewsUserServiceImpl();
    @Override
    protected void service(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        try {
            // 使用反射优化当前的servlet--可以将请求地址中的最后一个值作为被执行的方法
            // 1.获取请求地址
            String requestURI = req.getRequestURI();
            String[] split = requestURI.split("/");
            // 2.获取数组中最后一个值
            String flag = split[split.length - 1];
            System.out.println("flag = " + flag);
            // 3.使用反射读取同名方法
            Method method = NewsUserServlet.class.getDeclaredMethod(flag, HttpServletRequest.class,
                    HttpServletResponse.class);
            method.setAccessible(true);
            System.out.println("----这里的this对象指的是-------" + this);
            method.invoke(this, req, resp);
        } catch (NoSuchMethodException e) {
            throw new RuntimeException(e);
        } catch (IllegalAccessException e) {
            throw new RuntimeException(e);
        } catch (InvocationTargetException e) {
            throw new RuntimeException(e);
        }
    }
}













 
 
 
 
 
 









登录

下面是登录的具体实现代码

// 定义login方法--
// 先根据用户名查询用户是否存在 不存在返回用户名有误
// 存在的话判断解密之后密码是否正确 若正确返回包含用户id的token
protected void login(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
    System.out.println("login方法被执行");
    // 1.接收json数据 转换成java对象
    NewsUser user = WebUtil.readJson(req, NewsUser.class);
    System.out.println("转换的java对象为 = " + user);
    // 2.调取方法 验证用户名和密码
    NewsUser newsUser = service.findByUserName(user.getUsername());
    System.out.println("查询到的用户信息为 = " + newsUser);
    // 3.根据结果返回不同的数据
    Result result = null; // 定义一个封装结果的对象
    if (newsUser == null) {
        // 用户名不正确
        result = new Result(501, "用户名有误", null);
    } else { // 判断密码是否正确
        // 查询到的是加密的密码 加密之后比较MD5
        String pwd = MD5Util.encrypt(user.getUserPwd());
        if (pwd.equals(newsUser.getUserPwd())) {
            // 密码正确
            // 生成token
            String token = JwtHelper.createToken(newsUser.getUid().longValue());
            Map map = new HashMap(); //定义map是为了把token放到封装的结果集中的data中
            map.put("token", token);
            // 封装到结果集 result
            result = new Result(200, "success", map);
        } else {
            // 密码不正确
            result = new Result(502, "密码有误", null);
        }
    }
    // 4.把结果封装成Result 返回json格式的响应信息
    WebUtil.writeJson(resp, result);
}

public class NewsUserDaoImpl extends BaseDao implements NewsUserDao {
    @Override
    public NewsUser findByUserName(String uname) { //根据用户名查询用户信息
        String sql = "select uid,username,user_pwd userPwd,nick_name  nickName from news_user where username=? ";
        // 调取basedao中的basequery方法进行查询 先传如对象 再传入sql和传入的变量
        List<NewsUser> list = baseQuery(NewsUser.class, sql, uname);

        return list.size() > 0 ? list.get(0) : null;
    }
}

# 根据token获取用户信息

对应的NewsUserServlet的代码为：

// 根据token值查询用户信息
protected void getUserInfo(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
    // 1.获取请求头中的token
    String token = req.getHeader("token");
    // 2.验证token的有效性
    boolean bl = JwtHelper.isExpiration(token);
    Result result = null;
    if (bl) {
        // token异常
        result = new Result(504, "notlogin", null);
    } else {
        // toeken有效 获取token中的userid
        Long userId = JwtHelper.getUserId(token);
        // 调取service层 根据用户id查询用户信息
        NewsUser user = service.findById(userId.intValue());
        // 根据提供的信息把获取到的密码设为""
        user.setUserPwd("");
        Map map = new HashMap();

        // 这里注意一定要和要求的key相对应 否则后期很难找问题 这里的loginUser中的U是大写
        map.put("loginUser", user);

        result = new Result(200, "success", map);
    }
    // 把结果封装成Result 返回json格式的响应信息
    WebUtil.writeJson(resp, result);
}

# 注册

校验用户名--注册

在注册时，首先需要校验用户名是否已经被占用(这里可以直接调用上面登录的代码，根据返回的结果判断是否已经占用了用户名)，然后再进行注册 具体代码如下：

// 注册用户
protected void regist(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
    // 1.接收参数
    // 参数 key-value req.getParameter()
    // 参数json WebUtil.readJson()
    // 头信息传值 req.getHeader()
    NewsUser user = WebUtil.readJson(req, NewsUser.class);
    // 2.先查询用户是否存在 再调取注册用户的方法
    NewsUser byUserName = service.findByUserName(user.getUsername());
    Result result = new Result();
    if (byUserName == null) {
        // 用户名不存在 调取service层注册用户的方法
        service.insertUser(user);
        // 注册成功
        result.setCode(200);
        result.setMessage("success");
    } else { // 用户名被占用
        result.setCode(505);
        result.setMessage("用户名被占用");
    }
    // 3.根据结果返回不同信息给用户
    WebUtil.writeJson(resp, result);
}
