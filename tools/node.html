# Node

# NodeJs版本切换

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时环境，可以使 JavaScript 运行在服务器端。使用 Node.js，可以方便地开发服务器端应用程序，如 Web 应用、API、后端服务，还可以通过 Node.js 构建命令行工具等。
在 Node.js 中，我们可以使用 JavaScript 来编写服务器端程序，这也使得前端开发人员可以利用自己已经熟悉的技能来开发服务器端程序，同时也让 JavaScript 成为一种全栈语言。
首先在github上下载nvm，也就是Node.js的版本管理工具，点击下载open in new window
这个链接提供的是Windows版本，其余可以在github上搜索nvm下载在第一个下载即可，这里不再放链接，windows版本请下载nvm-setup.exe

注意

在安装nvm之前，可以把电脑中之前下载的nodejs卸载，不知道是否有影响
在安装过程中首先第一个选择的是nvm的安装位置，第二个选择的是nodejs的安装位置。

安装完成之后，win+r调用cmd命令行，输入 nvm ls 命令查看当前有哪些nodejs版本，可以看到目前还没有任何版本
比如我们使用nvm安装一个 16.20.1 的版本， 就可以输入 npm install 16.20.1 等待下载安装即可
安装其他版本类似，只需换掉后面的版本号即可
此时，再次输入 nvm ls 命令，可以看到，目前安装了两个版本

C:\Users\18032>nvm ls
    18.16.0
    16.20.1

提示

切换nodejs的版本,如下所示,可以看到切换成功

C:\Users\18032>nvm use 18.16.0
Now using node v18.16.0 (64-bit)

C:\Users\18032>node -v
v18.16.0

C:\Users\18032>npm -v
9.5.1

C:\Users\18032>nvm use 16.20.1
Now using node v16.20.1 (64-bit)

C:\Users\18032>node -v
v16.20.1

C:\Users\18032>npm -v
8.19.4

提示

nvm常用命令

nvm off                     // 禁用node.js版本管理(不卸载任何东西)
nvm on                      // 启用node.js版本管理
nvm install <version>       // 安装node.js的命名 version是版本号 例如：nvm install 8.12.0
nvm uninstall <version>     // 卸载node.js是的命令，卸载指定版本的nodejs，当安装失败时卸载使用
nvm ls                      // 显示所有安装的node.js版本
nvm list available          // 显示可以安装的所有node.js的版本
nvm use <version>           // 切换到使用指定的nodejs版本
nvm v                       // 显示nvm版本
nvm install stable          // 安装最新稳定版

提示

nvm设置镜像源

在nvm安装目录下找到setting.txt文件，在最后面添加

node_mirror: https://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/

# npm的使用

NPM全称Node Package Manager，是Node.js包管理工具，是全球最大的模块生态系统，里面所有的模块都是开源免费的；也是Node.js的包管理工具，相当于后端的Maven 。

# 配置

# 配置阿里镜像
npm config set registry https://registry.npm.taobao.org/
# 确认配置已生效 应输出 https://registry.npm.taobao.org/
npm config get registry
# 恢复默认的官方源
npm config set registry https://registry.npmjs.org/
# 配置全局依赖下载后存储位置
npm config set prefix "D:\GlobalNodeModules"
# 确认配置已生效
npm config get prefix
# 查看npm版本
npm -v
# 升级npm版本 
npm install -g npm@9.6.6

# 常用命令

# 项目初始化(2选1)
npm init
npm init -y
# 安装依赖(查看所有[依赖地址](https://www.npmjs.com))
# 默认安装在cmd窗口指定事务前缀路径
npm install 包名
npm install 包名@版本
# 安装全局依赖包(安装到d:/GlobalNodeModules)则可以在任何项目中使用它，而无需在每个项目中独立安装该包。
npm install -g 包名
# 安装package.json中的所有记录的依赖
npm install
# 升级依赖
npm update 包名
# 卸载依赖
npm uninstall 包名   /   npm uninstall -g 包名
# 查看项目依赖
npm ls
# 查看全局依赖
npm  list  -g
# 运行命令
npm run ...
