# 内连接与外连接

MySql连接查询——内连接与外连接open in new window

这里我们使用两张表用户表(t_customer)和订单表(t_order)来作示例,建表语句以及表中的数据如下所示

CREATE TABLE `t_customer` (
`customer_id` INT NOT NULL AUTO_INCREMENT, 
`customer_name` CHAR(100), 
PRIMARY KEY (`customer_id`) );

INSERT INTO `t_customer` VALUES (1, 'c01');
INSERT INTO `t_customer` VALUES (2, 'c02');
INSERT INTO `t_customer` VALUES (3, 'c03');

CREATE TABLE `t_order` ( 
`order_id` INT NOT NULL AUTO_INCREMENT, 
`order_name` CHAR(100), 
`customer_id` INT, 
PRIMARY KEY (`order_id`) ); 

INSERT INTO `t_order` VALUES (1, 'o1', 1);
INSERT INTO `t_order` VALUES (2, 'o2', 2);
INSERT INTO `t_order` VALUES (3, 'o3', 4);

表中的数据如下所示:

mysql> select * from t_customer;
+-------------+---------------+
| customer_id | customer_name |
+-------------+---------------+
|           1 | c01           |
|           2 | c02           |
|           3 | c03           |
+-------------+---------------+
3 rows in set (0.00 sec)

mysql> select * from t_order;
+----------+------------+-------------+
| order_id | order_name | customer_id |
+----------+------------+-------------+
|        1 | o1         |           1 |
|        2 | o2         |           2 |
|        3 | o3         |           4 |
+----------+------------+-------------+
3 rows in set (0.00 sec)

内连接

使用INNER JOIN......ON连接两个表，会略过为空的行，这里的INNER可以不写，默认就是内连接，但是写上可读性更好

SELECT
	c.customer_name,
	o.order_name 
FROM
	t_customer c
INNER JOIN 
	t_order o 
ON 
	c.customer_id = o.customer_id 

# 结果显示如下: 
+---------------+------------+
| customer_name | order_name |
+---------------+------------+
| c01           | o1         |
| c02           | o2         |
+---------------+------------+
2 rows in set (0.00 sec)

左外连接

使用LEFT JOIN，（等价于LEFT OUTER JOIN，OUTER可以忽略），意思是把LEFT JOIN左边的表作为主表，右边的表作为副表
左表的记录将会全部表示出来，而右表只会显示符合条件的记录，右表不符合条件的记录均为NULL

SELECT
	c.customer_name,
	o.order_name 
FROM
	t_customer c
LEFT JOIN 
	t_order o 
ON 
	c.customer_id = o.customer_id

# 结果显示如下: 
+---------------+------------+
| customer_name | order_name |
+---------------+------------+
| c01           | o1         |
| c02           | o2         |
| c03           | NULL       |
+---------------+------------+
3 rows in set (0.00 sec)

右外连接

使用RIGHT JOIN，（等价于RIGHT OUTER JOIN，OUTER可以忽略），意思是把RIGHT JOIN右边的表作为主表，左边的表作为副表
右表的记录将会全部表示出来，而左表只会显示符合条件的记录，左表不符合条件的记录均为NULL

SELECT
	c.customer_name,
	o.order_name 
FROM
	t_customer c
RIGHT JOIN 
	t_order o 
ON 
	c.customer_id = o.customer_id

# 结果显示如下: 
+---------------+------------+
| customer_name | order_name |
+---------------+------------+
| c01           | o1         |
| c02           | o2         |
| NULL          | o3         |
+---------------+------------+
3 rows in set (0.00 sec)
